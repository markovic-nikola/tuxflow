Capabilities

Everything in one place

Stop juggling terminal tabs. TuxFlow gives you a unified view of everything running in your project.

▶

Process Management

Start, stop, and restart dev servers. Auto-restart with exponential backoff on crashes.

✧

AI Agents

Run Claude Code, Codex, Gemini CLI, OpenCode, and other agents side-by-side with your processes.

🔔

Agent Idle Alerts

Get notified the moment an agent finishes its turn — terminal BEL detection plus an optional silence-based fallback, with per-agent sound overrides.

☰

Multi-Project

Open multiple projects in one window with expandable sidebar sections and drag-and-drop.

→

SSH Connections

Connect to remote hosts from ~/.ssh/config. Managed like any other process.

>_

Embedded Terminals

Full VTE4 terminals with true color, mouse events, search, and clickable URLs.

⚙

Stack Detection

Auto-detects Node.js, Rust, Go, Python, PHP, Docker and suggests commands.

⚙

Git Integration

Status, diff, commit, push, and pull from a built-in dialog. Live ahead/behind indicators and the current branch in the status bar.

▒

Resources

Per-process CPU and memory via /proc, aggregated across child process trees. See what's consuming resources at a glance.

☍

MCP Server

Unix socket API so AI agents can list processes, read logs, and control your environment.

🔊

Notifications

Per-project title/icon, focus-aware suppression, and bundled notification sounds with picker and preview.

☯

Theming

Dark, light, and system themes with accent colors and terminal color schemes.

Getting started

Install TuxFlow

Pick the method that suits your distro.

# Download the latest .deb from GitHub Releases
sudo dpkg -i tuxflow_*_amd64.deb
sudo apt-get install -f

Download from GitHub Releases

# Download the latest .tar.gz from GitHub Releases
tar xzf tuxflow-*-x86_64-linux.tar.gz
cd tuxflow-*-x86_64-linux
./install.sh              # installs to /usr/local (requires sudo)
./install.sh --user       # installs to ~/.local (no sudo)

Download from GitHub Releases

# Ubuntu/Debian
sudo apt install libgtk-4-dev libadwaita-1-dev libvte-2.91-gtk4-dev build-essential

# Fedora
# sudo dnf install gtk4-devel libadwaita-devel vte291-gtk4-devel gcc

# Arch
# sudo pacman -S gtk4 libadwaita vte4

git clone https://github.com/markovic-nikola/tuxflow.git
cd tuxflow
cargo build --release
./target/release/tuxflow

Integration

MCP Server

Let AI agents observe and control your dev environment.

  • List and monitor running processes
  • Read terminal output and logs
  • Start, stop, and restart processes

Claude Code config

{
  "mcpServers": {
    "tuxflow": {
      "command": "tuxflow-mcp",
      "args": ["my-project"]
    }
  }
}
